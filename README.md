# Predição de Clientes com Machine Learning

## Sobre o projeto

Aplicação desenvolvida com Python, Scikit-Learn e Streamlit para realizar previsões utilizando Machine Learning.

## Tecnologias

- Python
- Pandas
- Scikit-Learn
- Streamlit

## Funcionalidades

- Interface interativa
- Previsão em tempo real
- Modelo treinado com Machine Learning
- Visualização simplificada

## Como executar

```bash
pip install -r requirements.txt
streamlit run app.py
